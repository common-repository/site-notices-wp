Show notices, alert, popup, info messages on your WordPress site easily. Choose where to display notice on any single page/post/product or on complete site.

== Description ==

This plugin allows you to display different kind of notifications to your site users, or show special information or alert messages on any specific page/post/product.

**Features:**

* Display site wide notices
* Display notices on any specific page/post/product
* Choose from any four pre-defined color schemes/notification types
* Extremely customizable, set your own color, icon, title and message
* Display notices on six screen positions
* Two animation styles
* Instantly preview notices, to see how it looks on front-end
* Easily customizable through CSS


== Installation ==

1. Upload the plugin directory to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Go to **WordPress Admin Dashboard > Notices > Add New**

== Frequently Asked Questions ==

= Can I customize notification color? =

Yes, you can choose from four pre-defined colors or set your own custom color.

= Does notifications has icons? =

Yes, there are four pre-defined icons for success/info/warning & error.

= Can user hide the notification =

Yes, user can hide the notification, also admin can set to hide the notification automatically after few seconds.

= Is the add-on is compatible with the Latest version of WordPress? =

Yes, all our add-ons are compatible with latest version of WordPress.
